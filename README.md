# React + Vite: Personal Porfolio Website

Responsive Personal Portfolio

Description: A clean, responsive personal website designed to showcase a developer's skills, projects, and contact information, featuring a custom theming engine.

    Tech Stack: React (Vite), React Router DOM, Context API, CSS Variables.

    Key Features:

        Dark/Light Mode: Implemented a global theme toggle using Context API and CSS variables for persistent user preference.

        Routing: Client-side navigation between Home, About, Projects, and Contact pages without page reloads.

        Form Validation: Custom logic and Regular Expressions (RegEx) to validate email formats and required fields before submission.

        Responsive Design: Built with pure CSS Grid and Flexbox to ensure mobile and desktop compatibility without external UI frameworks.
